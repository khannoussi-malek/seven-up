# Documentation 

### 0.Requirements :
Node v18+ & npm

### 1.Quick start : 
Install dependecies : 
```
npm install
```

Run :
```
node app.js
```

---
### Api List

get current user : 
```
GET http://localhost:80/users/{user_id}
```
Login 
```
POST  http://localhost:80/users/login
```
Add User (Register) : 
```
POST  http://localhost:80/users/
```
